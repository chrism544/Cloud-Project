# Archived Page Builder

This directory contains the archived CraftJS page builder implementation that was removed due to persistent drag-and-drop issues.

## Contents

- `craftjs/` - Complete CraftJS implementation with widgets, components, and utilities
- `CRAFTJS_*.md` - Documentation files detailing implementation attempts and debugging
- `CRAFTJS_*.txt` - Debug logs and verification files

## Issues Encountered

1. Drag and drop functionality never worked properly despite multiple fix attempts
2. Complex component architecture made debugging difficult
3. CraftJS library had compatibility issues with the React/Next.js setup

## Archived Date

December 2024

## Replacement

The page builder has been replaced with a simple HTML editor for immediate functionality while a new custom solution is developed.
